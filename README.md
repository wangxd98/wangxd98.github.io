# Xiaodong Wang · Personal Academic Page

This project is Professor Wang Xiaodong's personal academic webpage, featuring a simple and steady academic style, suitable for GitHub Pages or personal academic showcases. The page clearly presents Professor Wang Xiaodong's personal profile, research interests, academic achievements, and honors.

## ✨ Page Features

- **Structured Academic Information**：Includes modules such as personal profile, academic positions, major honors, representative papers, research projects, monographs and teaching materials, and references.
- **Visual Identity Tags**：Uses badges and keyword tags to highlight research areas, titles, and academic positions.
- **Image Display Support**：A placeholder area is reserved at the top for easy insertion of personal photos or academic-related images.
- **Responsive Design**：Displays well on both desktop and mobile devices, with a clear information hierarchy.
- **Based on Reliable Data**：Content is compiled from publicly available academic records, with references noted.
